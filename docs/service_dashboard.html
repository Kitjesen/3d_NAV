TelemetryService gRPC :50051
▶ StreamFastState Server Stream 20-60Hz
高频遥测流：位姿、速度、IMU、关节角度（16 DOF）、定位质量评分、速度缩放因子。
FastState 包含：pose (x,y,z,quat), twist (linear,angular), imu (acc,gyro), joint_positions[16], localization_score, speed_scale
▶ StreamSlowState Server Stream 1-2Hz
低频状态流：当前模式、任务摘要、系统资源（CPU/内存/电池）、网络质量、话题频率、健康状态。
SlowState 包含：mode, active_task, resource_usage, topic_rates[], health_status (13 项子系统检查), geofence_status
▶ StreamEvents + AckEvent Server Stream + RPC
事件流（支持回放）：系统启停、导航事件（到达/失败/阻塞）、安全告警（急停/碰撞/倾斜）、传感器故障、定位丢失、任务事件、网络状态变化。
AckEvent 标记事件已读。

Launch Modes

Mapping

建图模式 — SLAM + 避障，手动遥控驾驶
  • LiDAR Driver
  • SLAM (Fast-LIO2 / Point-LIO)
  • Terrain Analysis + Local Planner
  • Robot Driver
  • gRPC Gateway
  • Semantic (optional)
ros2 launch navigation_bringup.launch.py slam_profile:=fastlio2

Navigation

导航模式 — 预建地图 + ICP 定位 + 自主导航
  • LiDAR Driver
  • SLAM + ICP Localizer
  • PCT Global Planner + Adapter
  • Terrain Analysis + Local Planner
  • Robot Driver (han_dog_bridge)
  • gRPC Gateway
  • Semantic (optional)
ros2 launch navigation_run.launch.py map_path:=/path/to/map

Exploration

探索模式 — 无预建地图，语义感知 + 前沿探索
  • LiDAR Driver
  • SLAM (实时建图)
  • Terrain Analysis + Local Planner
  • Semantic Perception (YOLO + CLIP)
  • Semantic Planner (VLN)
  • Robot Driver
  • gRPC Gateway (optional)
ros2 launch navigation_explore.launch.py target:="找到餐桌"
