## 🎯 核心技术亮点

### 1. Fast-Slow双进程架构
```
Fast Path (70%场景):
  场景图匹配 → 多源置信度融合 → ~10ms响应

Slow Path (30%场景):
  ESCA过滤(200→15物体) → LLM推理 → ~2s响应
```

### 2. 多源置信度融合
```python
fused_score = (
    0.35 × 标签匹配 +
    0.35 × CLIP相似度 +
    0.15 × 检测器置信度 +
    0.15 × 空间关系
)
```

### 3. ESCA选择性Grounding
```
200个物体 → 关键词匹配 → 1-hop扩展 → 区域扩展 → 15个物体
减少90% tokens，大幅降低API费用
```

### 4. 性能优化
- TensorRT INT8量化
- CLIP特征缓存 (60-80%命中率)
- 动态类别缓存
- 批处理推理
